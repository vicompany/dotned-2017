VueJS Basics: data & props

Data

  • Defined inside the component
  • The component can manipulate its own data

Props

  • Passed from a parent component to a child component
  • The child component can't manipulate the props
  • The parent component can manipulate the props
  • The props are updated inside the child component

VueJS Basics: Instance

  • An application is started with one Vue instance.
  • Often refered as the vm, for view model
    (remember, depending on the view model, the DOM is manipulated).

VueJS basics: Bindings

  • events:
    Easily bind click, input, etc events on elements
  • classes:
    Add or remove classes based on the data
  • states:
    Display or hide elements based on the data
  • model:
    Bind a form input directy to a data property
